## How to run

### Installation

```bash
  npm i
```

### Run Answer 1 Script
```bash
  ts-node answer1.ts [array parameter separated by ',']
```
#### Excample 
```bash
  ts-node answer1.ts 1,3,5,4,2,8,5,7
```
#### Output
```
  [8,7,6,5,4,3,2,1]
```

### Run Answer 2 Script
```bash
  ts-node answer2.ts [array parameter separated by ','] [size]
```
#### Excample 
```bash
  ts-node answer2 -3,4,0,-2,6,-1 2
```
#### Output
```
  5
```
